# Slava Likh Portfolio Site

Static portfolio prototype. No build step, no paid hosting required.

## Open locally

Open `index.html` in a browser.

## Free hosting options

- GitHub Pages
- Netlify free site
- Vercel free static deployment

No custom domain is required for the first version.

## Structure

- `index.html` - portfolio homepage
- `cases/` - individual case pages
- `assets/` - local font and CV PDFs
- `styles.css` - screen and print styles
